#### **Οδηγίες**
- Οι οδηγίες **SEMGENT** & **ENDS** καθορίζουν μια ομάδα από δεδομένα ή εντολές που θέλουμε να τοποθετήσουμε μαζί σε ένα τμήμα. Τα δεδομένα ή οι εντολές που περιλαμβάνονται μεταξύ αυτών λέγονται λογικό τμήμα (logical segment). -- Πριν απο ENDS βαζεις MOV AH,4Ch INT 21h.

	- ASSUME CS: είναι απαραίτητη σε κάθε πρόγραμμα .ΕΧΕ
	- DS & ES (DATA & EXTRA) είναι απαραίτητες πριν την χρήση εντολών που έχουν σχέση με μεταφορά δεδομένων από και προς δ/νσεις μνήμης.

### LAB 3
#### Εντολή μεταφοράς MOV
- σύνταξη: **MOV** προορισμός, προέλευση
- οι τελεστές πρέπει να είναι ίσου πλήθους bit δηλαδή ή και οι δύο 1 byte ή και οι δύο 1 word = 2 bytes.
#### Πρόσθεση - Αφαίρεση
- **ADD** & **SUB** προορισμός, προέλευση
- όπως και στην MOV, απαγορεύεται η χρήση καταχωρητών τμημάτων
#### Εντολές Αύξησης Μείωσης Τελεστών
- **INC** τελεστής   //αύξηση κατά 1
- DEC τελεστής   //μείωση κατά 1
#### Πολλαπλασιασμός
- **MUL** προέλευση
- Αν ο τελεστής είναι ενός byte τότε: 
		αν AL=10 και DL=15
		MUL DL
		τότε AX=150
- Αν είναι δύο byte (1 word) τότε η προέλευση πολλαπλασιάζει το περιεχόμενο του ΑΧ και το αποτέλεσμα αποθηκεύεται σαν διπλή λέξη στους καταχωρητές DX:AX.
#### Διαίρεση
- **DIV** προέλευση
- Αν ο τελεστής είναι ενός byte τότε:
		αν ΑΧ=115 και DL=10
		DIV DL
		τότε AH=5 και AL=11
- Πηλίκο στον ΑL και υπόλοιπο στον AH
- Αν είναι δύο byte τότε η προέλευση διαρεί το περιεχόμενο της διπλής λέξης που είναι αποθηκευμένο στους DX:AX και το μεν πηλίκο της διαίρεσης αποθηκεύεται στον ΑΧ το δε υπόλοιπο στον DX
#### Εμφάνιση 
**Παράδειγμα String**
- στο **τμήμα δεδομένων** δηλώνουμε το αλφαριθμητικό ως εξείς:
	Message db 'The result is: ', 10, 13, '$'
	όπου $ είναι ο τελευταίος χαρακτήρας του μηνύματος
- στο **τμήμα κώδικα** μεταφέρουμε στον καταχωρητή DX την ενεργό διεύθυνση του αλφαριθμητικού που θέλουμε να τυπώσουμε
```
	LEA DX, message --Ή-- MOV DX, offset message
	MOV AH, 09h
	INT 21h
```
**Παράδειγμα Αριθμού**
- στο τμήμα του κώδικα αυτή τη φορά μεταφέρουμε το περιεχόμενο στον καταχωρητή DL
	MOV DL, num1
- στο τμήμα του κώδικα μετατρέπουμε τη δυαδική τιμή του αριθμού σε τιμή του πίνακα ASCII
	ADD DL, 30h
- στο τμήμα του κώδικα καλούμε τη 2η κλήση του DOS αμέσως μετά για εμφάνιση ενός χαρακτήρα
	MOV AH, 02h
	INT 21h

### LAB 4
#### Δείκτες Κατάστασης (6 bits)
- **CF (Carry Flag)** Χρησιμοποιείται σαν ένα επιπλέον δυαδικό ψηφίο σε αριθμητικές εντολές (ADD, SUB, ADC, SBC). Αν προκύψει ένα κρατούμενο ή απαιτηθεί δανεισμός παίρνει τιμή 1 αλλιώς 0.
- **SF (Sign Flag)** Χρησιμοποιείται από αριθμητικές ή λογικές πράξεις. Εάν το αποτέλεσμα είναι θετικό γίνεται 0, εάν είναι αρνητικό γίνεται 1
- **ZF (Zero Flag)** Γίνεται 1 αν το αποτέλεσμα μιας αριθμητικής ή λογική πράξης είναι 0, αλλιώς μένει 0
- **OF (Overflow Flag)** Γίνεται 1 όταν το προσημασένο αποτέλεσμα μιας πράξης (σε συμπλήρωμα του δύο) είναι πολύ μεγάλο ή πολύ μικρό για να χωρέσει στον τελεστή του αποδέκτη

---
- **CMP** συγκρίνει τον προορισµό µε την προέλευση ενηµερώνει τους δείκτες κατάστασης(Flags) ανάλογα µε το αποτέλεσµα της πράξη
#### Διακλάδωση χωρίς συνθήκη
- **JMP** : Εντολή διακλάδωσης του προγράμματος σωρίς συνθήκη
- πχ για επανάληψη εντολών:
	kapou:
		JMP kapou
- πχ για αποφυγή εκτέλεσης εντολών:
	JMP next
	...
	...
	next:
#### Διακλάδωση με συνθήκη 

| Εντολή  | Διακλάδωση              | Δείκτες          |
| ------- | ----------------------- | ---------------- |
| JB/JNAE | Μικρότερο μη προσ.      | CF=1             |
| JAE/JNB | Οχι μικρότερο μη προσ.  | CF=0             |
| JBE/JNA | Οχι μεγαλύτερο μη προσ. | CF=1 ή<br>ZF=1   |
| JA/JNBE | Μεγαλύτερο μη προσ.     | CF=0 KAI<br>ZF=0 |
| JE/JZ   | Ίσα                     | ZF=1             |
| JNE/JNZ | Άνισα                   | ZF=0             |

### LAB 5
#### LOOP
- η LOOP μειώνει τον **CX** κατά 1 και εάν η τιμή του > 0 πραγματοποιείται άλμα στην ετικέτα, αλλιώς εκτελείται η εντολή που βρίσκεται μετά την LOOP.
*Παράδειγμα*
```
	DATA SEGMENT
		minima DB 'ΤΕΛΟΣ ΠΡΟΓΡΑΜΜΑΤΟΣ$'
	DATA ENDS

	MOV CX, 18 ;metrhths epanalhpshs
	LEA BX, minima ;energos diefthynsh ths metavlhths minima ston bx

	loop-ektyp:
		MOV DL, [DX] ;ston dl o xarakthras pros emfanhsh -- byte.
		INC BX ;afjhsh deikth
		MOV AH, 02H ;klhsh DOS gia emfanish
		INT 21h ;diakoph DOS
		loop loop-ektyp ;epanalhpsh an CX>0
```
#### LOOPE & LOOPNE
- used after CMP
- LOOPE: loop when C>0 and ZF=1 (equal)
- LOPPNE: C>0 and ZF=0 (not equal)
